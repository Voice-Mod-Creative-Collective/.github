## üöÄ Download & install (archive delivery)
1. Click the **Start NOW** button below ‚Äî the toolkit archive will begin downloading automatically (a single `.zip` or `.tar` file).
2. Once downloaded, extract the archive to a convenient folder (e.g., `C:\VoiceMod-Enhance-Toolkit` or `~/VoiceMod-Enhance-Toolkit`).
3. Open the extracted folder and locate the installer file (e.g., `Install_VoiceModTools.exe`).
4. Double-click the installer and follow the on-screen prompts.
5. After installation, launch VoiceMod and import templates via **File ‚Üí Open** or apply presets in the Effect panel, then reload assets.
> Tip: For effects, organize audio files in a `media/` subfolder and relink clips in VoiceMod if paths change.
